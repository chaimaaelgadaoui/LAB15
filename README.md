# 📱 LAB15 - Gestion des Étudiants

Application Android développée en Java utilisant SQLite pour gérer les étudiants.

## 🚀 Fonctionnalités

* Ajouter un étudiant
* Rechercher un étudiant par ID
* Supprimer un étudiant
* Affichage des logs avec Logcat

## 🛠️ Technologies

* Java
* Android Studio
* SQLite

## 📂 Architecture

* `MainActivity` : interface utilisateur
* `EtudiantService` : logique métier (CRUD)
* `MySQLiteHelper` : gestion base SQLite
